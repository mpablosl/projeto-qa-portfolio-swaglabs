# projeto-qa-portfolio-swaglabs
Praticando O Dia a Dia de um QA: A Prática de Testes Manuais Funcionais - Curso na DIO - Formação Quality Assurance (QA) Experience# 🧪 Projeto QA - Portfólio SwagLabs

Este repositório contém um projeto prático de testes de software (QA), com foco na validação de funcionalidades de uma loja virtual (SwagLabs).

O objetivo deste projeto é demonstrar conhecimentos em análise de requisitos, criação de cenários e casos de teste, execução de testes e reporte de bugs utilizando boas práticas de QA.

---

## 📌 Conteúdo do Projeto

- 📖 User Stories
- ✅ Critérios de Aceite
- 🧪 Cenários de Teste
- 📋 Casos de Teste
- 🐞 Registro de Bugs
- 📎 Evidências de Testes (PDFs)

---

## 🛠️ Ferramentas Utilizadas

- Jira (gestão de tarefas e testes)
- Git e GitHub (versionamento)
- Testes manuais

## 🎯 Objetivo

Aplicar na prática conceitos de QA como:

- Levantamento e entendimento de requisitos
- Escrita de User Stories
- Definição de critérios de aceite
- Criação de cenários e casos de teste
- Execução de testes manuais
- Identificação e reporte de bugs

---

## 🚀 Diferenciais

- Estrutura baseada em fluxo real de QA
- Organização de artefatos por tipo
- Simulação de ambiente ágil (Scrum/Kanban)
- Documentação clara e objetiva
